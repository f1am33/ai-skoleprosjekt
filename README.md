# Stone Howl - Rock Band Website

En moderne, mørk og edgy nettside for rockebandet Stone Howl.

## Funksjoner

- **Responsivt design** som fungerer på alle enheter
- **4 hovedseksjoner:**
  - Hjem med hero-bilde og preview-kort
  - Konserter med kommende shows og billettlinker
  - Musikk med diskografi og streaming-lenker
  - Om oss med bandhistorie og medlemmer
- **Mørkt og edgy design** med røde aksenter
- **Interaktiv navigasjon** med smooth scrolling
- **Mobilvennlig** med hamburger-meny og touch-gestures
- **Animasjoner** og hover-effekter
- **Keyboard-navigasjon** (piletaster for seksjoner)

## Teknologier

- HTML5 med semantisk struktur
- CSS3 med moderne features (Grid, Flexbox, Animations)
- Vanilje JavaScript for interaktivitet
- Responsive design for mobil/tablet/desktop

## Bruk

1. Åpne `index.html` i en nettleser
2. Naviger mellom seksjoner via menyen
3. Klikk på lenker for å se interaksjoner
4. Test på mobil for responsivt design

## Tilpasning

- Bytt ut bilde-URLs med egne bandbilder
- Oppdater konserter, musikk og bandinfo
- Juster farger i CSS-variablene
- Legg til egne sosiale medier-lenker

## Filsstruktur

```
├── index.html     # Hovedstruktur
├── styles.css     # All styling
└── script.js      # Interaktiv funksjonalitet
```
